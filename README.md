# Yacht Tracker Streamer

本システムは、洋上作業における現在地のリアルタイム記録、動的なV地点（測定点）の管理、およびCSVテンプレートを用いたエリア確認を目的とした業務用Webアプリケーションです。

## ✨ 主な機能と特徴

* **リアルタイム座標同期 (Socket.IO):**
    * 操作者の端末で入力・記録した座標やCSVデータを、別端末（閲覧者）へリアルタイムに同期・共有します。
* **動的な測定点（V地点）管理:**
    * V1〜V3を標準とし、必要に応じてV4、V5...と動的に入力枠を追加・削除可能。
* **高度なCSV連携:**
    * **ベースCSV (青枠線):** 作業エリアの境界線（edge）を描画。
    * **確認用CSV (黄色ピン):** 過去の測定点などを参考として配置。
    * CSV出力時は、テンプレートの構造（edgeやway_order）を一切壊さず、入力済みのV座標だけをway行として自動最適化して保存します。
* **現場特化の省電力GPSロジック:**
    * リアルタイム表示ON時は、端末負荷を抑えるため「3秒間隔」での位置取得・軽量アイコン描画を実施。
    * バッテリー保護のため、現在地表示は「1時間で自動OFF」になります。
    * 追跡OFF時でも、V地点のボタンを押した瞬間だけ「単発でGPSを取得」して記録するエコ仕様です。
* **安心の自己修復機能:**
    * ブラウザのローカルストレージを活用し、誤って画面をリロードしたりサーバーが再起動したりしても、読み込み済みのCSVやピンの状態を自動復旧します。

## 🛠️ 画面操作マニュアル

### 参加モード
* **[操作者モード] (赤字):** 座標の記録、手動入力、CSVの読み込み、リセット操作の全権限を持ちます。
* **[閲覧者モード] (青字):** 状況のモニタリングと、自身の端末上でのみ完結するCSVクリア・データ保存が可能です。（入力操作などはロックされます）

### 3種類のリセット機能（誤操作防止対策済み）
現場での状況に合わせて、消去する範囲を選べます。
1.  **⚠️ 全リセット (上部):** 追加したV地点、読み込んだCSV、座標など、全てを完全に初期化します。
2.  **🗑️ CSVクリア (中央):** 現在記録中のV座標は残したまま、読み込んだ「青枠線」や「黄色ピン」だけを画面から消去します。
3.  **🗑️ 座標リセット (下部):** 読み込んでいるCSV（エリア情報）は残したまま、記録したV地点の座標と追加枠だけを初期化し、次の測定に備えます。

### 手動入力モード
操作者モード時、タイトル下のスイッチをONにすることで、GPSを使用せず手動で座標（テキスト）を入力・修正できるテスト用モードが解放されます。

## 🚀 起動方法

### サーバー起動 (Node.js)
\`\`\`bash
cd pos-server
npm install
node server.js
\`\`\`
※Windows環境の場合は \`start_system.bat\` を実行してください。

### アプリへのアクセス
* サーバーを立てているPC自身から見る場合: `https://localhost:3000/portal.html`
* 同じWi-Fi（またはTailscale等のVPN）に繋がっている別端末から見る場合: `https://<サーバーPCのIPアドレス>:3000/portal.html`
